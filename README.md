# Le Fournil de Lhoussaine — Site statique

Ce dépôt contient un petit site statique pour une boulangerie : page d'accueil, histoire, liste des pains avec descriptions et images.

Fichiers inclus
- `index.html` — page principale
- `css/styles.css` — styles
- (Images référencées depuis Unsplash — remplacez-les si besoin)

Personnalisation rapide
- Remplacez les textes (nom, histoire, contact) dans `index.html`.
- Pour utiliser vos propres images : créez un dossier `images/` et remplacez les URLs dans les balises `<img src="...">` par `images/nom-image.jpg`.
- Ajustez les couleurs dans `css/styles.css` (variables : `--accent`, `--bg`, etc.).

Déploiement (option simple)
- Hébergement statique : GitHub Pages
  1. Créez un dépôt et poussez ces fichiers à la racine.
  2. Dans les réglages du dépôt, activez GitHub Pages depuis la branche `main` (ou `gh-pages`) et le dossier `/` (root).
  3. Le site sera disponible en quelques minutes à `https://<votre-utilisateur>.github.io/<votre-repo>/`.
